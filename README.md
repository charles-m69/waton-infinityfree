# waton-infinityfree
This is a Content Management System (CMS) academic project in a mockup of an E-commerce website. For the Web Programming subject in Olivarez College - Tagaytay. Parodied from the store Watsons. Utilized CRUD operations with MySQL, and authentication process using PHP. The following is demonstrated by the site:
- Public facing page
- Item categories and product page
- Search and filter items
- Simple log in and register with CAPTCHA mockup and password change
- Admin dashboard: Manage categories and items
- Admin role level: Editor, Admin, Superadmin
- Manage admin: Change role level, profile info, and password
- Change images for user interface: Carousel Banners, Categories, Site Icon, etc.
- Downloadable XML for items list in dashboard

Frontend
- HTML5
- CSS3
- Bootstrap 5 | Framework and Icons

Backend
- PHP
- JavaScript
- AJAX

Database & Server
